# SHOUTING-MACHINE

## USE

Allows user to enter text in a form and returns it in uppercase. Will allow you to edit input without exiting form. 

## SET-UP/INSTALL

Open file and enter text you want uppercased in the form box, click button. Resubmission of text functions without refresh.
